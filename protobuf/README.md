# protobuf ｃ语言

## 依赖protobuf
git clone https://github.com/protocolbuffers/protobuf.git　　

cd protobuf　　　

./configure --prefix=/usr/　　

make　　

sudo make install　　




## protobuf-c 编译
git clone https://github.com/protobuf-c/protobuf-c.git　　

cd protobuf-c/　　

./configure --prefix=/usr/　　

make　　

sudo make install
